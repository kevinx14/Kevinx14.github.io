vi多行注释及取消注释

本方法在LINUX MINT下通过，理论支持Linux。 vi多行注释 使用方向键将光标移动至需要注释的首行行首或尾行行首，按 Ctrl + v 进入v模式 按方方向键选着需要注释的行首 按大写 I 键 (Shift+i) 输入 # 或 // 按 ESC 键返回 vi取消注释 使用方向键将光标移动至需要取消注释的首行行首或尾行行首，按 Ctrl + v 进入v模式 按方方向键选着需要取消注释的行首 按 d 键删除注释符号
