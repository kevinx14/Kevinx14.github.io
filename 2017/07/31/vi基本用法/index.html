本方法在LINUX MINT下通过，理论支持Linux。

  • vi打开文件

如文件不存在则自动新建。

  • vi添加内容

使用方向键移动光标至需要位置按 a 键或 i 键，必须为小写状态。此时进入 输入模式 可以输入内容了。在终端做下会出现
-- 插入 --
或
-- INSERT --
输入结束后按 ESC 键退出 输入模式 。

  • vi删除内容

使用方向键移动光标至需要位置按 DEL 键删除内容。

  • vi保存与提出文件

按 : 键，终端底部出现 : 与 光标 ，则表示已经进入 命令模式 输入 wq ，然后回车保存并退出文件。
w 表示保存
q 表示退出
